# nailsalon Noël

横浜・元町中華街のネイルサロン **nailsalon Noël** 公式サイト。

- 公開サイト: https://monyakaanyanya-stack.github.io/nailsalon-noel/
- Instagram: [@nailsalon.noel](https://www.instagram.com/nailsalon.noel/)

## 構成

静的サイト（HTML / CSS / JS のみ、ビルド工程なし）。GitHub Pages で公開しています。

| パス | 内容 |
|---|---|
| `index.html` ほか | 日本向けサロン公式サイト |
| `shop/` | 海外向け販売サイト（英語） |

`git push` すると GitHub Pages に自動反映されます（数分）。
